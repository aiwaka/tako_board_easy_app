## 概要

うさぎのお世話記録をつける掲示板.

- メールアドレスを利用したログイン認証機能.
- リモートデータベースを用いた記録.
- 携帯端末でも見やすい.
- Typescript を用いた実装.

## 注意

現在新規登録等は不可能.
タイムラインのような機能もない.

## 使用技術

- Firebase（Firestore, Authentication）
- Vue.js 3（Composition API）
- Typescript
- SCSS
